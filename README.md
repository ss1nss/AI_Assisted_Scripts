AI-Assisted Workplace Automations

Python scripts developed with AI-tools to automate repetitive heavy-load tasks in a financial services environment. Each script eliminated manual, error-prone processes — reducing processing time and improving accuracy across supplier management, client onboarding, and data integration workflows.

Scripts

Supplier Data Consolidation — Merges and aggregates supplier information across multiple Excel datasets, reconciling legacy terms and pricing against updated client agreements to streamline onboarding.
Invoice Lookup Tool — Queries exported bank system reports to surface invoice data by invoice number, enabling faster and more accurate responses to supplier inquiries.
Fintech Upload Mapper — Extracts and maps operational data fields from source Excel files into a fintech platform upload template, eliminating manual data entry and reducing upload errors.
Tools Used: Python · AI-tools (like Copilot) · openpyxl · pandas
